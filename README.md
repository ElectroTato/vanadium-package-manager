# Vanadium Package Manager
Vanadium is a simple package manager made for the minecraft mod [ComputerCraft: Tweaked](https://modrinth.com/mod/cc-tweaked/), please note that this is meant to be more of a personal project to make it easier for me to port stuff over into my worlds without having to rely on wget or pastebin, I can't guarentee that it will be usable.

# Adding your own packages
If you wanna add your own packages to Vanadium then feel free to do so!

Just create a pull request with your package added! (Please do note that if you project uses multiple files you should make your main.lua an installer).
